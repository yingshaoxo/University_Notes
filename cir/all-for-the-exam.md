#### 10、万用表量程的选择，数字万用表二极管档的使用。
量程的选择: 被测物的最大值应是满刻度的三分之二。如测量20V的电压，应选用30V的量程。

测二极管: 把量程转换到`欧姆档R*100`或`R*1K`。黑表笔接二极管正极，红表笔接二极管负极，正向电阻越小越好。红表笔接正极，黑表笔接负极，电阻无穷大为好。

#### 11、鸟牌功率计的结构、使用方法与场合。
结构: 电源、主机、传感器

使用场合: 测量正向和逆向平均功率、峰值，或突发功率、电压驻波比(VSWR)、振幅因数，及互补累计分布函数(CCDF)等，用来对发射机及天馈系统的性能进行检测和故障判断

使用方法: 

1. 开机
2. 设定标度
3. 设定元件型号
4. 开启射频源，读数
5. 长按OK键2秒，睡眠。长按OK键8秒，完全关闭设备

#### 12、天馈线分析仪使用场合、端接部件、测试结果分析与故障判断。
使用场合: 快速、准确地提供天馈线分析，如插入损耗/电缆损耗、回波损耗/驻波比、故障定位

端接部件: 开路/短路器、50欧姆端接、天线

故障判断:

1. 终端接了 50欧姆标准负载: abs(回波损耗) < 30dB
2. 终端接了 天线: 14dB < abs(回波损耗) < 20dB
3. 终端接了 开路 或 短路: 0dB < abs(回波损耗) < 5dB

#### 16、CIR设备GSM-R话音单元、数据单元、主控单元的功能。
话音单元:

1. 在主控单元的控制下自动完成GSM-R`网络注册`
2. 在主控单元的控制下自动完成GSM-R`话音调度通信`
3. 在主控单元的控制下自动完成GSM-R`机车功能号的注册注销`
4. 在主控单元的控制下自动完成GSM-R`车次功能号的注册注销`

数据单元:

1. 在主控单元的控制下自动完成GSM-R`网络注册`
2. 在主控单元的控制下`完成IP地址的获取`
3. 在主控单元的控制下`完成GPIS IP地址的申请`
4. 在主控单元的控制下`通过GPRS网络采用UDP/IP方式，发送无线车次号校核信息，接收无线调度命令，并发送回执信息`

主控单元：

1. 实现对MMI、记录单元、GPS单元、GSM-R话音单元、GSM-R数据单元、450MHz单元、LBJ单元、接口单元的通信与控制
2. 接收转发处理各单元的数据，并转发至记录单元
3. 完成了对多包调度命令信息的拼包处理并让MMI显示
4. 在 GSM-R 工作模式下，根据车次号，控制无线车次号校核信息的发送
5. 实现对各路音频的汇接
6. 完成GSM-R调度通信的语音业务
7. 完成其他数据业务

#### 17、多频段合路器的功能。

#### 18、450M无线列调系统频率配置。
4频组配置。

同频通话时，机车台和车站台收发频率都定为F4(低频)；

异频通话时，机车接收频率为F2，发射频率为F4(高频频率)，站台相反。

F1、F3也是高端频率，主要在“大三角”通信时使用。

#### 19、机车数据采集编码器功能。

# Resource

1. 调制中，待传输的基带信号也称`调制`信号；高频振荡信号也称`载波`信号；加载了基带信号的高频信号也称`已调`信号。
2. `半双工制`通信方式主要用于专业移动通信系统中，如汽车调度系统、铁路调度系统等。无线列车调度通信系统中采用同频单工方式时，相邻车站电台之间会存在`同频`干扰。
3. 振子的全长为2 l。若对称振子的全长为`0.5 λ`，称半波对称振子。当振子`半径a`越大，输入阻抗随l /λ变化越缓慢，可以使天线在较宽的频率范围内满足匹配。天馈线系统驻波比`为1`时，表示完全匹配，在移动通信系统中，一般要求驻波比<`1.5` 。
4. 5010B型功率传感器正常配置中，正向功率探头功率值是反向功率探头的`10`倍。如果需要测量被测设备的额定功率，可将功率计输出端接`50`欧的标准假负载。
5. MMI操作显示终端分为`横向`式、`竖立`式两种外形结构，采用`嵌入`式安装方式。送受话器分为`紧凑`式送受话器和`通用`式送受话器两种规格。
6. `GSM-R数据`单元在主控单元的控制下完成IP地址的获取。`记录`单元具有对话音、承载业务信息及操作过程进行记录存储、话音回放的功能。`卫星定位`单元为CIR设备各单元提供标准时钟。
7. 目前全路450 MHz无线列调通信以`B`制式为主。 `f1、f3`频率是高端频率，主要在“大三角”通信时使用。
8. CIR/LBJ检测设备包含`CIR/LBJ 出入库检测设备`、`便携式机车设备测试仪`、`便携测试台`、`数据管理器`。
9. LBJ进入库检状态后，语音提示“请按键试验”，此时按下“`报警`”、“`列尾排风`”、“`风压查询`”、“`确认/签收`”键，LBJ将自检结果发至便携测试台和CIR/LBJ出入库检测设备。
10. 处理CIR故障时维修人员应遵循一“询问”、二“查看”、三“`思考`”、四“动手”的基本原则。
11. `信号调制`可以将要发送的信号变为已调载波。已调载波信号在经过`变频`成为射频载波信号。
12. 铁路450M列车无线调度系统中，语音信息采用`FM`调制，而调度命令、车次号校核数据信息采用`FFSK`调制；在铁路数字移动通信系统GSM-R中，则采用`GMSK`调制。
13. 当发射源与接收体之间存在相对运动时，接收体接收到的信息频率与发射源的信息频率不相同，这种现象称为`多普勒效应`。
14. 馈电系统主要涉及两个问题；一是天线与馈线的`阻抗`问题；二是天线与同轴线连接时的`从不平衡到平衡`问题。
15. GPS导航系统是以全球`24`颗定位人造卫星为基础，GPS的应用都是基于两个基本服务：`空间位置`服务和`时间`服务。 
16. 选择`二极管`档位可以读出二极管偏压值，使用指针万用表测量二极管应该选择`欧姆`档位和`二极管`档位。
17. 鸟牌功率计适用的5010B型功率传感器，可测量`正向`和`逆向功率`，因此可计算并显示`电压驻波比和其他匹配测试值` 。
18. CIR的GSM—R话音单元采用功率为`8w`的模块，载频间隔`200 khz`，双工频率间隔`45 MHz` ，参考灵敏度优于`-104dbm`，射频连接器为`TNC`。
19. 我国铁路技术规范规定，机车天线的极化方式为`垂直`极化。
20. 调度员与司机之间的450M通信，车站电台发射频率采用交替配置，车站电台发射频率为`f1、f2、f3`，接收频率为`f4`。
21. 机车数据采集编码器兼容`FFSK` 和`RS-422`两种方式。插在`TAX2`内的编码器，不断地捕获广播的机车运行数据信息
22. LBJ一种是LBJ和`司机控制盒`配套使用，另一种是和`CIR`配套使用。
23. 便携式机车设备测试仪开机后必须进入`测试机车设备`状态，才能对CIR等机车设备进行测试。
24. 兆欧表俗称摇表，是用来测量被测设备的`绝缘`电阻和`高值`电阻的仪表。
25. 无线通信的工作方式主要有`单工`通信、`双工`通信和`半双工`通信三种。
26. 天线的参数`增益`是用来衡量天线在某一特定方向收发信号的能力，它是选择基站天线最重要的参数之一。相同的条件下，其数值越高，电波传播的距离越远。
27. 同轴电缆馈线是`不平衡`的，而对称振子天线本身是一个平衡系统，当两者相连接时，会产生不平衡，因此需解决`从不平衡到平衡`的转换问题。
28. 副控状态下MMI上的`呼叫`、`复位` 和`主控`键有效，其余按键均无效。在副控模式下，按`主控` 按键3秒钟后，进入主控工作模式下的主界面。
29. 动车组给CIR主机设备直流供电电压是`110`V。
30. 持续按下MMI的`报警`键3s以上（或按`报警`键后3s内松开，随后5s内按下`确认/接受`键），MMI向LBJ发送启动报警命令。
31. 电缆布线时，需避开机车主电路电缆，保护距离应在`300`mm以上。
32. 机车综合无线通信设备关机，机车综合无线通信设备进行车次功能号的自动`注销`。
33. 当铁路专用无线通信设备发生障碍时，维护人员应及时准确的作出判断，迅速组织修复，努力缩短障碍时间。对影响行车安全的设备必须采取`倒、代`等应急措施，减少影响程度。
34. 观察分析法主要通过查看CIR各单元状态，特别是`灯显`，是故障处理的常用方法。
35. 在无线列车调度通信系统中，当采用`同频单工`方式时，相邻车站电台之间会存在同频干扰。
36. 在`自动切换`模式下，CIR将根据地理位置信息自动在450MHz和GSM-R两种工作模式间切换。
37. 标识特定旅程的列车车次号码，一般情况下在行程中该号码应`唯一`。运行超过`24`小时的车次，其车次号允许重复。
38. 障碍处理必须做到“五清”即`时间`清、`地点`清、`影响范围`清、`原因`清、`处理过程`清。
